StackJS 1.1 usage

Method Return Description
.clear() {null} Clears all elements from the Stack.
.exile({String} alias) {*} matched item Removes an item from the Stack and return it, matching by given alias
.get({String} alias) {*} matched item Returns some item matching by given alias.
.put({*} object, {String} alias*, {String} option) {Boolean} operation success Puts some item in the Stack. The alias must be unique and is mandatory, if the alias is duplicate, the item will be updated. Option can be 'nooverwrite' to don't update the item if the alias already exists.
.remove({String} alias1, {String} alias3, ..., {String} aliasN) {Number} amount of deletions Deletes items from the Stack using the given names.
.reset() {null} Clears all elements and states from the Stack.
.restore({String} state_name|{Number} state_number) {null} Restores a previous saved state, Can be restored by its name or number. If none parameter is specified it restores the last valid state.
.save({String} state_name) {null} Saves the current state of the Stack, the name is optional.
.size() {Number} amount of items Get the amount of items in the Stack.
