StackJS 1.4 usage

Method Return Description
.clear() <null> Removes all items from Stack, but don't resets it states.
.exile(<*> alias) [<*> matched item|<undefined>] Removes an item from the Stack and returns it, matching by given alias.
.get(<String> alias) [<*> matched item|<undefined>] Returns some item matching by given alias, but don't remove it from the Stack.
.put(<*> object, <String> alias*, <Boolean> overwrite) <Boolean> operation success Puts some item in the Stack. The alias must be unique and is mandatory, if the alias is duplicate, the item will be updated. If third parameter is set to false, repeated alias will not be overwrote.
.remove(<*> alias1, <*> alias2, ..., <*> aliasN) <Number> amount of deletions Deletes multiple items from the Stack and returns the number of deletions.
.reset() <undefined> Clears all elements and states from the Stack.
.restore([<String> stateAlais|<undefined>]) <Boolean> operation success Restores a previous saved state. If no alias is defined, restores the last save point.
.save([<String> stateAlais|<undefined>]) <undefined> Saves the current state of the Stack. If no alias is specified, it will saves on the top, otherwise it will save new or overwrite that alias.
.size() <Number> amount of items Returns the size of the Stack (a.k.a. Number of elements).
