Version 1.1

21/11/2010

Production release

New Features:

  • register() method now is called put()
  • new remove() method to remove N items from the Stack at one time
  • new reset() method to clear all items from the stack and the save points
  • new size() method to get the size of the Stack (number of items)

Version 1.0 beta

21/11/2010

Initial release

Features:

  • register() method to register something in the stack
  • get() method to get something from the stack
  • save() method to save a state of the stack
  • restore() method to restore a previous saved state of the stack
  • exile() method to remove some item from the stack and get it
  • clear() method to clear all items from the stack
